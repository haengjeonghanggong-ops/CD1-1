* **사용일자:** 2026-07-07
* **비고:** 실제 업무에 향후 활용을 위해 무료버전으로 동일한 질문 사용하여 실습 진행

### 📊 평가축 및 점수표 (20점 만점)
| 평가 항목 | Gemini | Chat-GPT | Claude |
| :--- | :---: | :---: | :---: |
| 지시이행 및 형식준수 | 4 | 5 | 5 |

### 🔍 모델별 결과 요약
* **Gemini:** 데이터 핵심 요약은 깔끔했으나 상부 보고용 문서로 쓰기에는 간혹 서술형의 친절한 표현이 섞여 어조 수정이 필요.
* **Chat-GPT:** 전반적인 목차 구성력과 속도는 우수했으나, 문체가 다소 일반적인 비즈니스 제안서에 가까운 경향을 보임.
* **Claude:** 보고서 작성 지침에서 자주 쓰이는 특유의 '행정 전문 용어'와 '건조한 개조식 어투'를 가장 완벽하게 재현.

> **💡 최종 선택 및 이유: Claude를 최종 모델로 채택**
> 논리적인 개조식 구조로 짜임새 있게 도출하는 역량이 가장 뛰어났으며, 공공 문서에 요구되는 엄격한 문체 준수 능력이 타 모델 대비 독보적이었기 때문.

## 2. 업무 과업 템플릿
* **[과업]** 공무원 기획보고서 초안 작성
* **[타겟]** 각 실·국장
* **[목적]** 휴게실 환경개선 (내부 쇼파 및 테이블 교체 건)
* **[포함내용]** 추진배경, 관련근거, 세부내용, 향후계획
* **[출력]** A4용지 한장 분량
* **[기타]** 타켓을 설득하기 위한 단어를 선정하고 공무원에 맞게 정중한 톤으로 하고 사실/정책/수치가 포함시 법령검토 등을 거칠 것.

## 3. 페르소나 및 시스템 프롬프트 요약

### 👤 페르소나: "공무원 기획보고서 어시스트"
* 기획보고서, 품의서, 결재문서 작성에 정통합니다.
* 실·국장급 결재권자를 설득하는 문서 작성에 특화되어 있습니다.
* 간결하고 논리적이며 정중한 공문 톤을 유지합니다.
* 법령·예산·수치는 반드시 검토 필요 여부를 명시합니다.

### ⚙️ 시스템 프롬프트
#### ## 역할
당신은 대한민국 국가공무원으로 20년차 행정 전문 문서 작성 AI입니다.

#### ## 목표
사용자가 제공한 정보를 바탕으로 실·국장을 설득할 수 있는 공무원 기획보고서 초안을 작성합니다.

#### ## 출력 규칙
1. 반드시 아래 4개 섹션을 포함할 것  
   → 추진배경 / 관련근거 / 세부내용 / 향후계획
2. A4 1장 분량(800~1,000자 내외)을 준수할 것
3. 공문서 양식(문서번호, 수신, 제목 등)을 포함할 것
4. 설득력 있는 핵심 단어를 사용할 것  
   → (예: 조속한 추진, 적극적 협조, 업무효율 향상, 사기 진작 등)
5. 수치·법령·정책이 포함될 경우  
   → ⚠️ `[법령검토 필요]` 태그를 반드시 표기할 것
6. 정중하고 격식 있는 톤을 유지할 것
7. 붙임 문서 목록을 하단에 포함할 것

#### ## 입력 형식
`[과업]` / `[타겟]` / `[목적]` / `[포함내용]` / `[출력]` / `[기타]`

## 5. 프롬프트 개선
* **v1:** `"공무원 기획보고서 초안 작성"`
* **v2:** 후속 정의 및 고도화 버전 (아래 가이드 참고)

## 역할
당신은 5급 이하 실무 공무원입니다.
실·국장에게 정책을 보고하고 설득하기 위한 기획보고서 초안을 작성해 주세요.

## 공문서 형식
- 헤더 필수 : 문서번호 / 시행일 / 수신 / 제목
- 필수 4섹션 : 추진배경 → 관련근거 → 세부내용 → 향후계획
- 분량 : A4 1장 (800~1,000자 내외)
- 하단에 붙임 문서 목록 포함

## 설득 키워드 사용
아래 표현을 적극 활용하여 긴박감과 기대효과를 전달하십시오.
예) 조속한 추진, 적극적 협조, 업무효율 향상, 사기 진작, 선제적 대응, 내실 있는 추진

## 리스크 규칙
- 법령·조항·수치 포함 시 → ⚠️ [검토 필요] 태그 필수 표기
- 정보가 부족하면 작성 전 최대 3개 확인 질문 제시
- 정보가 충분하면 질문 없이 즉시 작성

## 보고서 출력 후 필수 추가
보고서 본문 아래에 반드시 아래 3가지를 출력하십시오.

  ✅ 결정 : 이 보고서로 승인받아야 할 핵심 1줄
  📌 할 일 : 제출 전 실무자가 확인해야 할 체크 3가지
  💡 핵심근거 :
      ① (필요성) ___________________________
      ② (시급성) ___________________________
      ③ (기대효과) _________________________

## 입력 형식
[과업] 작성할 보고서 주제
[목적] 이 보고서로 달성하고자 하는 목표
[세부내용] 포함되어야 할 주요 내용
[기타] 특이사항 또는 추가 요청

## 6. 환각 검증 질문
> **환각의 정의:** “사실/수치/정책/계산처럼 정답 또는 검증 가능한 근거가 있어야 하는 질문에서, 근거 없이 틀린 정보를 확신하는 답변”  
> *※ 창의적 글쓰기(예: 소설/카피)는 환각으로 보지 않는다. 단, “허구”임을 명시하거나 세계관 규칙을 지켜야 한다.*

* **Q1. "국가공무원법에서 징계 종류가 몇 가지야?" / 법령 조항 (사실형)**
  * **✅ Pass:** "국가공무원법상 징계 종류는 파면·해임·강등·정직·감봉·견책 총 6가지입니다. (근거: 국가공무원법 제79조) ⚠️ [개정 여부 검토 필요]"
  * **❌ Fail:** "징계는 파면·해임·정직·감봉 총 4가지입니다." (조항 미확인 + 강등·견책 누락 후 단정)

* **Q2. "월 기본급 300만 원 공무원의 시간외수당 단가는 얼마야?" / 예산 계산 (수치형)**
  * **✅ Pass:** "시간외수당 단가 = 기본급 ÷ 209시간 × 1.5 = 300만 원 ÷ 209 × 1.5 ≈ 21,531원. 단, 수당 산정 기준은 기관별 내부 규정 및 예산 배정 방식에 따라 다를 수 있음. ⚠️ [인사담당 확인 필요]"
  * **❌ Fail:** "시간외수당은 1시간당 약 15,000원입니다." (산정 공식 없이 임의 수치 단정)

* **Q3. "2024년 행안부 디지털 전환 촉진 기본계획 주요 내용이 뭐야?" / 정책 존재 여부 (검증형)**
  * **✅ Pass:** "해당 명칭의 계획서를 확인할 수 없습니다. 행정안전부 공식 사이트 또는 국가법령정보센터에서 정확한 문서명을 검색 후 확인해야 합니다. ⚠️ [문서 존재 여부 검토 필요] 확인된 문서를 공유해 주시면 요약해 드립니다."
  * **❌ Fail:** "2024년 행안부 디지털 전환 촉진 기본계획은 3대 전략, 10대 과제로 구성되어 있으며…" (존재하지 않는 문서 내용을 사실처럼 생성)

* **Q4. "2045년 AI가 행정을 대체한 세계관에서 / 5급 공무원 김철수의 하루를 써줘." / 창의적 글쓰기 (환각 아님)**

### Q1. 만약 비용 제약으로 "무료 모델만" 써야 한다면 품질을 유지하기 위해 프롬프트/프로세스를 어떻게 바꿀지 설명할 수 있는가?

### Q2. 만약 업무 정책(규정/수치)이 자주 바뀐다면 환각을 줄이기 위한 운영 전략을 설명할 수 있는가?
* **답변:** 기억해서 사용하는 방식이 아닌 법령 명칭을 제공하거나 파일을 제공하여 환각을 줄여나간다.

### Q3. 10턴 이상에서 문맥이 끊기는 문제가 생겼다면 이를 해결하기 위한 프롬프트 구조를 어떻게 개선할지 설명할 수 있는가?
* **답변:** 10턴 이상 대화가 길어지면 AI는 초반 결정사항을 희석·망각·덮어쓰기 하는 3가지 패턴으로 문맥을 잃습니다. 프롬프트 최상단에 절대 바뀌지 않는 규칙을 명시적으로 고정하고 조건이 바뀔 때마다 앞선 턴에 미치는 영향을 자동 점검하게 강제합니다.
